# the-runner

*** HOW TO INSTALL ***

SYSTEM REQUIREMENTS

Windows, MacOS, or Linux

Python 3.6.0 or newer

IDLE (Integrated Development and Learning Environment)

PyGame

RUNNING THE GAME

1. Open TheRunner.py

2. Run code through shell in IDLE

3. Enjoy!
